### 2. tasizan.py の新規作成
- **ファイル**: `SRC/tasizan.py`
- **機能**: 1から50まで順に足していくプログラム
- **表示形式**: 累計を随時表示

#### 初期実装
```python
total = 0
for i in range(1, 51):
    if i == 1:
        print(i)
    else:
        print(f"{total}+{i}={total + i}")
    total += i
```

#### 修正履歴
1. **日本語記号への変更**
   - `+` → `＋`
   - `=` → `＝`

2. **語尾の追加**
   - 各計算結果の最後に「です」を追加

#### 最終実装
```python
total = 0
for i in range(1, 51):
    if i == 1:
        print(i)
    else:
        print(f"{total}＋{i}＝{total + i}です")
    total += i
```

## 実行結果例
```
1
1＋2＝3です
3＋3＝6です
6＋4＝10です
...
1225＋50＝1275です
```
